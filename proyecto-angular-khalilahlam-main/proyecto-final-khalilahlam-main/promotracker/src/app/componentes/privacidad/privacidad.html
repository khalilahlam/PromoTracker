Política de Privacidad

Esta aplicación, PromoTracker, ha sido desarrollada por Khalil Ahlam Karoui como proyecto académico de 2º DAW en el IES Salvador Gadea. El propósito de esta aplicación es educativo.

Datos personales

PromoTracker es un proyecto académico y no recoge datos personales reales de los usuarios. Toda la información gestionada dentro de la aplicación es simulada y utilizada solo con fines educativos.

Uso de la información

Cualquier dato dentro de la aplicación se utiliza únicamente para demostrar funcionalidades de gestión de promociones y cupones. No se almacena ni comparte información personal fuera de este proyecto.

Seguridad de los datos

Aunque PromoTracker no maneja datos reales, se recomienda siempre practicar buenas medidas de seguridad al desarrollar aplicaciones. Este proyecto sirve únicamente para fines educativos y no representa un sistema en producción.

Consentimiento

Al utilizar esta aplicación, aceptas que los datos son simulados y que el proyecto es únicamente académico.
